# xcloud-cli

## **安装(install):**
```
npm init -y
```
```
yarn add -D xcloud-cli
```

## **使用(usage)：**

### **新建模版(init)**
```
npx xcloud init
```

### **开发(dev)**
```
npx xcloud dev
```

### **打包(build)**
```
npx xcloud build
```


## **配置(config)：**
查看目录下xcloud.config.js文件
